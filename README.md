# 🌧️ TouchTheRain

基于 Qt 的桌面交互式动画小程序，模拟雨滴下落并支持鼠标交互。

---

## 📖 项目简介

**TouchTheRain** 是一个运行在桌面上的雨滴动画效果程序，窗口采用透明背景，实现雨滴持续下落的视觉效果。

当鼠标触碰到雨滴时，雨滴会立即向下移动一段距离，增强了趣味性和动态反馈。

---

## ✨ 功能特点

* 🌧️ 多雨滴循环下落动画
* 🖱️ 鼠标交互：触碰雨滴会产生位移
* 🎲 随机生成雨滴位置与速度，效果更自然
* 🪟 透明窗口，实现桌面覆盖效果
* ⚡ 基于定时器的实时刷新，运行流畅

---

## 🛠️ 技术实现

* **开发语言**：C++
* **开发框架**：Qt（Qt6）
* **核心技术点**：

  * 使用 `QTimer` 实现动画循环更新
  * 使用 `QRandomGenerator` 生成随机位置与速度
  * 重写鼠标事件（`mousePressEvent` / `mouseMoveEvent`）实现交互
  * 使用全局坐标（`globalPosition`）实现平滑拖拽
  * 基于状态更新（位置累加）实现连续运动效果
  * 使用透明窗口属性实现桌面悬浮效果

---

## 📦 使用方法

1. 下载 Releases 中的可执行文件（`.exe`）
2. 运行程序
3. 移动鼠标触碰雨滴，观察交互效果
